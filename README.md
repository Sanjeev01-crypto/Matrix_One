# Matrix_One 🚀

## 📌 Overview
Matrix_One is a high-performance Image Quality Assessment (IQA) model developed for competitive benchmarking.

## 🧠 Key Features
- Lightweight deep learning architecture
- Custom hybrid loss (MSE + Pearson + Rank Loss)
- Optimized for leaderboard performance

## 📊 Results
- Strong validation performance
- Competitive Kaggle-style submission

## 📁 Repository Structure
- `matrix (1).ipynb` → Training & inference pipeline
- `submission.csv` → Final predictions
- `Matrix_One.pdf` → Detailed project report

## ⚙️ Tech Stack
- Python
- PyTorch
- NumPy / Pandas
